Oncilla
=======

Introduction
------------

**Oncilla** is the documentation building package of `Manager <http://github.com/KelSolaar/Manager>`_, `Umbra <http://github.com/KelSolaar/Umbra>`_, `sIBL_GUI <http://github.com/KelSolaar/sIBL_GUI>`_, `sIBL_Reporter <http://github.com/KelSolaar/sIBL_Reporter>`_.

Installation
------------

The following dependencies are needed:

-  **Tidy** http://tidy.sourceforge.net/

This package is not meant to be installed but used as a git submodule repository.
